INSTALLATION STEPS:
-------------------

STEP 1: EXTRACT FILES
----------------------
Extract all files to your web directory:
- XAMPP: C:/xampp/htdocs/jobsite/
- WAMP: C:/wamp64/www/jobsite/
- Linux: /var/www/html/jobsite/
- Hosting: /public_html/jobsite/

STEP 2: CREATE DATABASE
------------------------
1. Open phpMyAdmin: http://localhost/phpmyadmin
2. Create new database: "jobsite_db"
3. Select the database
4. Click "Import" tab
5. Choose file: database.sql
6. Click "Go" button
7. Wait for success message

STEP 3: CONFIGURE DATABASE
---------------------------
1. Open file: config.php
2. Update these lines:
   
   define('DB_HOST', 'localhost');     ‚Üê Your DB host
   define('DB_NAME', 'jobsite_db');    ‚Üê Your DB name
   define('DB_USER', 'root');          ‚Üê Your DB username
   define('DB_PASS', '');              ‚Üê Your DB password

3. Save the file

STEP 4: CHANGE ADMIN PASSWORD
------------------------------
1. Open file: config.php
2. Find this line:
   
   define('ADMIN_PASSWORD', 'admin123');

3. Change 'admin123' to your secure password
4. Save the file

STEP 5: TEST YOUR WEBSITE
--------------------------
Open your browser and visit:

‚Üí Homepage: 
  http://localhost/jobsite/public/index.php

‚Üí Admin Panel: 
  http://localhost/jobsite/public/admin.php
  Password: (the one you set in Step 4)

TROUBLESHOOTING:
----------------

ERROR: "Database connection failed"
FIX: Check database credentials in config.php

ERROR: "Page not found"  
FIX: Make sure all files are in the correct folder

ERROR: "Admin login not working"
FIX: Clear browser cache, check password in config.php

ERROR: "Newsletter not working"
FIX: Check browser console (F12) for JavaScript errors
